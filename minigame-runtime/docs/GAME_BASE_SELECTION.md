# Game Base Selection

本文件吸收自 `MINIGAME_V4_Starter_Pack.zip`，用于后续筛选开源小游戏底座。

## 当前原则

当前项目已有可运行 H5 原型，因此短期不替换底座。

底座筛选只用于后续商业化和复用评估，不允许为了调研下载大量仓库或打断当前主线。

## 底座优先级

1. **Phaser / H5**：快速验证底座。
2. **Cocos Creator**：商业上线候选底座。
3. **Godot / VN / WebGAL**：叙事结构参考，不作为第一桶金主线。

## 评分表

| 维度 | 分数 | 说明 |
|---|---:|---|
| 许可证 | 20 | 是否允许商用，是否有传染性限制 |
| 平台适配 | 20 | H5 / 微信 / 抖音适配难度 |
| 代码清晰度 | 15 | 是否容易让 AI / 人类维护 |
| 广告结构 | 15 | 是否容易接激励广告、复活、解锁内容 |
| 存档结构 | 10 | 是否容易保存进度、解锁、数据 |
| UI 结构 | 10 | 是否适合 HUD / 监控 / 控制台 UI |
| 资源体积 | 5 | 是否适合小游戏包体限制 |
| 可换皮性 | 5 | 是否容易通过内容包 / skin 替换题材 |

## 分级

```text
85 分以上：主底座候选
70-84：备选
60-69：参考
60 以下：淘汰
```
