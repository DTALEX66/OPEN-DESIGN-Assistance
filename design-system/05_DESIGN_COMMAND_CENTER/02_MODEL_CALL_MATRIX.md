# 02｜MODEL CALL MATRIX

## GPT
产品判断、设计方向、UI结构、风格系统、去AI味、最终筛选、设计任务单、UI Schema 草案。

## DeepSeek
批量草案、多方向标题、多组事件、低成本发散、选题池、小红书标题池、抖音脚本初稿。

## Qwen
中文润色、小红书口吻、游戏日志、悬疑文本、按钮文案、系统提示。

## CC Switch
判断是否可实现、拆开发任务、防止过度设计、检查技术风险、判断是否该调用 Codex。

## Codex
代码实现、接入 UI Schema、读取 content-pack、按设计稿实现界面、绑定按钮、实现弹窗、修复 bug。

## Codex 限制
不自由设计、不决定玩法、不扩展任务、不自行改颜色、不自行重构 UI。
