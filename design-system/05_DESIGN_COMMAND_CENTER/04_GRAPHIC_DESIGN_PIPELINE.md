# 04｜GRAPHIC DESIGN PIPELINE

## 适用范围
小红书封面、抖音封面、游戏宣传图、招聘作品集、海报、长图、分享图、广告图、游戏首页 KV。

## 标准流程
1. Hermes 生成 Design Request
2. GPT 判断目标用户和视觉策略
3. DeepSeek 生成 20 个标题/卖点
4. Qwen 优化中文表达
5. GPT 筛选 3 个方向
6. Figma 制作版式
7. Photopea / Photoshop 处理主视觉
8. 输出多尺寸
9. Vision 审查
10. Hermes 归档最终版本

## 固定尺寸
小红书封面：3:4；抖音封面：9:16；分享图：5:4；横版宣传：16:9；作品集页面：A4 / 16:9。
