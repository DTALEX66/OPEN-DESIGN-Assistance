# 03｜DESIGN REQUEST SYSTEM

每个设计任务必须先有 Design Request。

```md
# DESIGN REQUEST

## Goal
要设计什么？

## Usage
用在哪里？
- 游戏主界面
- 小红书封面
- 抖音封面
- 广告弹窗
- 分享图
- 作品集页面

## Target User
给谁看？

## Style Information
风格关键词：
参考图：
禁用风格：

## Content
标题：
副标题：
按钮文案：
提示文案：

## Output
需要输出：Figma设计稿 / PNG / SVG / UI Schema / Codex实现说明

## Deadline
什么时候完成？

## Tag
Graphic / UI / Poster / Marketing / Game / Review
```

没有 Design Request，不进入设计。没有 UI Schema，不调用 Codex 实现游戏界面。
