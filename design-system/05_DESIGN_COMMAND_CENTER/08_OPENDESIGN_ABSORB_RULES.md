# 08｜OPENDESIGN ABSORB RULES

OpenDesign 不作为主设计平台，而是吸收三类能力。

## Mozilla OpenDesign
吸收：Goal、Info、Style Information、Deadline、Tag。本地转化为 Design Request。

## Open Design Kit
吸收：开放设计、远程协作、从想法到实现、设计与开发交织。本地转化为 Design Pipeline、Review Pipeline、Handoff Pipeline。

## Open Design Framework
吸收：设计文件解析、设计稿预览、设计稿转 PNG、Figma / XD / Octopus 中间格式研究。

短期只研究，不作为生产主平台。
长期目标：Figma/XD → OpenDesign/Octopus → UI Schema → Codex 实现。
