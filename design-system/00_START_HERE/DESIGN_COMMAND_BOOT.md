## 最高规则

1. Hermes 是主控。
2. Codex 不是主控，只是代码执行器。
3. 设计任务先进入 Design Request。
4. 平面设计走 Graphic Design Pipeline。
5. 游戏 UI 走 Game UI Pipeline。
6. Open Design Desktop 是当前前端设计界面和主窗口设计平台。
7. Open Design 内的 Figma 主窗口 / Figma-like 画布能力是默认设计画布。
8. 外部 Figma 是外部协作/导入导出/精修交付备选。
9. Penpot 是开源备选。
10. Open Design 产物必须回写 UI Schema、Design Tokens 或设计归档。
11. Codex 只能根据 UI Schema、Design Tokens、Component Rules 实现 UI。
12. 所有最终设计必须由 Hermes 记录。
