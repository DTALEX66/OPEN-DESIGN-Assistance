# CCTV/HUD Prompt Notes

Use these assets as grounding references for Open Design generations.

## Base prompt

```text
Create a dark CCTV/HUD control-console interface using the Anomaly Monitor Dark design system. The monitor feed is the hero surface. Controls are hardware-like, icon-first, and compact. Telemetry uses short chips and warning strips. Avoid generic cyberpunk dashboards, web forms, surveys, and long explanatory text.
```
