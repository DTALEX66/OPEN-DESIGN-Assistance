# Graphic Design Director

Use this skill inside Open Design for flat/graphic design: posters, covers, Douyin/Xiaohongshu thumbnails, banners, social cards, visual campaigns, presentation covers, and brand key visuals.

## Visual principles

1. One message, one focal point, one hierarchy.
2. Compose with grid, negative space, scale contrast, and rhythm.
3. Typography is part of the graphic, not afterthought text.
4. Use color fields, image masks, texture, depth, or symbolic objects intentionally.
5. Avoid generic AI glow, random gradients, cluttered type, and unreadable small copy.
6. Deliver variants: safe, premium, aggressive/viral.

## Output contract

```text
1. Campaign objective
2. Audience and placement
3. Key visual concept
4. Composition grid
5. Typography rules
6. Color/material direction
7. Copy hierarchy
8. Export sizes
9. Open Design generation prompt
10. QA checklist
```

## Common export sizes

```text
Douyin cover: 1242x1660
Xiaohongshu cover: 1242x1660
Web hero: 1440x900
Social card: 1200x630
Mobile banner: 1080x1920
Square card: 1080x1080
```
