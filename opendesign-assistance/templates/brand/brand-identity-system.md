# Brand identity system template

Use for brand design, 品牌设计, visual identity, logo direction, campaign style, product brand systems, and Open Design-native brand packs.

## Required brief

```text
brand: name and category
users: primary audience and decision maker
positioning: premium | mass | playful | official | tech | cultural | creator | industrial
competitors: 3-5 references to avoid/counter
usage: website | app | packaging | signage | social | exhibition | pitch deck
must keep: existing logo/color/type/assets
must avoid: banned colors, cliches, competitor similarity
```

## Identity components

```text
brand idea: one sentence
logo direction: wordmark | symbol | monogram | system mark | no-logo refresh
color system: primary, secondary, neutral, semantic
font system: display, body, code/data, CJK fallback
shape language: radius, grid, icon stroke, container rhythm
imagery: photo, render, illustration, texture, diagram
layout DNA: dense | editorial | modular | cinematic | utility | luxury
motion/sound if relevant
```

## Deliverables

```text
1. Brand essence
2. Visual territory: 3 directions
3. Tokens: color/type/spacing/radius/shadow
4. Logo/mark direction notes
5. Application examples: website, social, signage, deck
6. Do/don't rules
7. Open Design prompt
```

## Quality gates

- Identity must work in black/white.
- Chinese and English typography must both be considered.
- System needs application examples, not just mood words.
- Avoid direct imitation of reference brands.

## Open Design prompt add-on

```text
Create a brand identity system with positioning, visual territory, token system, typography, logo direction, application examples, and do/don't rules. Avoid moodboard-only output and direct imitation.
```
